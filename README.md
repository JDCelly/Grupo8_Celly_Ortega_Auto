# Grupo8_Celly_Ortega_Auto
Repositorio del grupo 8 para la evidencia de automatización 

## 🛠️ Proyecto de Software de Automatización

Este repositorio contiene el desarrollo del **Software de Automatización** realizado como parte de las actividades académicas del programa de **Ingeniería Mecatrónica**.

## 👥 Integrantes del equipo

- **Juan Celly**
- **Juan Ortega**

## 🎯 Objetivo del proyecto

Desarrollar herramientas y módulos de automatización que permitan optimizar procesos mediante la integración de software, electrónica y control, aplicando los conceptos aprendidos en el programa de Ingeniería Mecatrónica.

## 📁 Estructura del repositorio

- `/src` — Código fuente del proyecto  
- `/docs` — Documentación técnica y manuales  
- `/tests` — Pruebas y validaciones  
- `/assets` — Recursos adicionales (imágenes, diagramas, etc.)

## 🚀 Tecnologías utilizadas

- Python  
- Herramientas de automatización y control  
- Librerías específicas según cada módulo desarrollado  

## 📌 Estado del proyecto

En desarrollo. Las actualizaciones se realizarán de forma continua conforme avance el proyecto.
